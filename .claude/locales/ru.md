# Руководство по переводу на русский язык

## Общие принципы

- Придерживайся оригинала как можно ближе: не опускай обороты и уточнения из английского текста.
- Используй форму «мы» — читатель является соавтором, а не студентом.

## Перевод технических терминов

| Английский | Русский | Не использовать |
|------------|---------|-----------------|
| expose / expose a function | предоставлять / предоставить функцию | экспонировать |
| stateful | с состоянием / обладающий состоянием | состоятельный |
| sandboxed | изолированный | — |
| map (тип данных) | ассоциативный массив | — |
| map to / mapped to | преобразуется в | отображается в |

## Оформление

- Составные термины пишутся через дефис: «Lua-скрипт», «Elixir-приложение», «Lua-код».
- Английские аббревиатуры и названия (BEAM, NIF, API) не переводятся и не склоняются.
